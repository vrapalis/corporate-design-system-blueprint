# Introduction to the Atomic Design System

---

> The library based on [Angular-Material](https://material.angular.io/) and [Bootstrap 5](https://getbootstrap.com/)

<br/>

## How to start

<br/>

### Angular Material
- If you want to have **Angular Material** import style in your main scss like ``@import "libs/shared/ui/src/lib/styles/angular-material";`` <br/>

### Bootstrap 5
- If oyu want to have **Bootstrap 5** import style in your main scss like ``@import "libs/shared/ui/src/lib/styles/bootstrap-5.scss";`` <br/>
- And also you need to add scripts to scripts block in angular.json or project.json file.
``
"scripts": [
"node_modules/@popperjs/core/dist/umd/popper.min.js",
"node_modules/bootstrap/dist/js/bootstrap.bundle.js"
]
``
